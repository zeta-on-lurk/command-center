# DESIGN SYSTEM: PROJECT A.D.A.M.

**Theme:** Dark, stoic, authoritarian, minimalist, cinematic. It should look like an elite military or intelligence command center, not a standard school app.
**Tech Stack:** HTML5, Tailwind CSS (via CDN), Chart.js.

**Tokens:**
- Backgrounds: Deep black (`bg-zinc-950`).
- Cards: Dark grey with subtle thin borders (`bg-zinc-900 border border-zinc-800`). No soft drop shadows.
- Typography: 
  - Headings: Serif font ('Playfair Display' via Google Fonts) for authority.
  - Data/Numbers: Monospace font ('JetBrains Mono') for raw data look.
- Accents: Monochromatic (white/gray), with ONE harsh accent color (blood red `#8b0000` or cold cyan `#00b7eb`) used ONLY for critical alerts.
- Corners: Sharp or very minimal rounding (`rounded-sm`).
